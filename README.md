#how to run

1. Clone repo - ```git clone https://github.com/agentsmity/testSmartProject.git ~/test```
2. Make shure that you have composer v2
3. ```cd ~/test && composer install```
4. run command - ```./console app:report 2021-01-29 2021-01-31```
